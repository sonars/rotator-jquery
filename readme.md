### npm install

> `npm i rotator-jquery`

### yarn install

> `yarn add rotator-jquery`

### Usage

> HTML Template

 ```html
  <div class="rotator">
      <div class="item">First Item</div>
      <div class="item">Second Item</div>
      <div class="item">Third Item</div>
      <div class="item">Fourth Item</div>
  </div>
 ```
> jquery template

 ```javascript
 <script>
   $('.rotator').rotator();
 </script>
 ```
